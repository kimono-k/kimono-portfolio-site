My Projects.

Kimono Dating



A personal start-up idea for a dating website for anime, game and cosplay fans. I've built the Front-end for this project with HTML5, CSS3 and JavaScript. For the styling I've mainly used CSS flexbox, grid and media queries to make the website responsive. I still have future plans for this project to build a back-end system so people can chat with each other, make friends etc. I want to do this with the Python framework Django.

My Korean Drama



A hobby project where I've built an application to save my favourite Korean Drama's so I can talk with my friends about all the dramas I've watched.

Banjo-Tooie SFX



Banjo-Tooie is a game which I've used to play in my childhood. I wanted to learn how to work with EventListeners in JavaScript that's why I've built this project to have a deeper understanding of how they exactly work. You can press the corresponding buttons on your keyboard to activate the sound or you can mouse click. Have fun laughing about this web app.

List Pagination and Filtering
